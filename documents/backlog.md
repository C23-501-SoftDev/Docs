# Бэклог проекта (Product Backlog)

## Epic 4: Администрирование и Безопасность (Priority 1 - Инфраструктура)
*Без пользователей и пространств невозможно создавать документы.*
- [ ] **Task 4.1:** Инициализация проекта (Spring Boot, БД, настройка файловой SMB-системы + Git).
- [ ] **Task 4.2:** Создание сущности `User`, реализация авторизации и выдачи JWT-токенов.
- [ ] **Task 4.3:** Реализация RBAC (глобальные роли: Admin, Editor, Reader).
- [ ] **Task 4.4:** UI/API: Панель администратора (CRUD пользователей).
- [ ] **Task 4.5:** CRUD сущности `Space` (Пространства).
