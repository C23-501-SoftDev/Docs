# US4.1.4 Панель администратора (CRUD пользователей)

## Описание
Разработка административного интерфейса для управления учетными записями пользователей (создание, редактирование, удаление, блокировка) и назначения им ролей.

## Технические требования
* **Стек/Инструменты:** Java/Spring Boot (Backend API), Frontend (UI администратора).
* **Архитектурные аспекты:** API эндпоинты для администраторских операций CRUD над сущностью `User`.
* **Инфраструктура/Конфигурация:** Доступ к панели ограничен ролью `Admin`.

## Критерии приемки (Acceptance Criteria)

### Сценарий 1: Управление пользователем
* **Given:** Учетная запись администратора.
* **When:** Администратор создает или обновляет карточку пользователя в панели.
* **Then:** Изменения (роли, статус) успешно сохраняются в БД и применяются при следующем входе пользователя.

### Сценарий 2: Удаление пользователя
* **Given:** Учетная запись пользователя существует.
* **When:** Администратор удаляет учетную запись.
* **Then:** Доступ пользователю к системе блокируется/учетная запись удаляется.
