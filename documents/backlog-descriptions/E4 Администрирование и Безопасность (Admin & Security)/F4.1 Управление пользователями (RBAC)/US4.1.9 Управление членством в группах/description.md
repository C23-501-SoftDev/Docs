# US4.1.9 Управление членством в группах

## Описание
Возможность добавления пользователей в группы для управления правами доступа.

## Технические требования
* **Backend:** API для добавления пользователя в группу и удаления из нее.

## Критерии приемки (Acceptance Criteria)

### Сценарий 1: Добавление пользователя в группу
* **Given:** Существуют пользователь и группа.
* **When:** Администратор добавляет пользователя в группу.
* **Then:** Пользователь становится членом группы и наследует связанные с ней политики доступа.

### Сценарий 2: Удаление пользователя из группы
* **Given:** Пользователь состоит в группе.
* **When:** Администратор удаляет пользователя из группы.
* **Then:** Пользователь теряет права доступа, полученные через данную группу.
