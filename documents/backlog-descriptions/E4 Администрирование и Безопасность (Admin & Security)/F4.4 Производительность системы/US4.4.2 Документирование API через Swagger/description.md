# US4.4.2 Документирование API через Swagger/OpenAPI

## Описание
Внедрение OpenAPI (Swagger) для автоматической генерации документации по API системы.

## Технические требования
* **Tools:** `springdoc-openapi-starter-webmvc-ui`.
* **Configuration:** Настройка эндпоинта `/swagger-ui.html` для просмотра документации.

## Критерии приемки (Acceptance Criteria)

### Сценарий 1: Доступность документации
* **Given:** Приложение запущено.
* **When:** Администратор/разработчик переходит по ссылке `/swagger-ui.html`.
* **Then:** Отображается интерактивная страница со списком всех REST API эндпоинтов.
