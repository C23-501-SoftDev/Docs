# US4.3.1 Рассылка Email-уведомлений

## Описание
Внедрение подсистемы рассылки Email-уведомлений для информирования пользователей о важных событиях в системе (изменение документа, предоставление прав доступа).

## Технические требования
* **Стек/Инструменты:** Java/Spring Boot (Spring Mail/JavaMailSender), SMTP-сервер.
* **Архитектурные аспекты:** Служба уведомлений (Async Notification Service). Асинхронная отправка писем для предотвращения блокировки основного потока.
* **Инфраструктура/Конфигурация:** Настройка параметров SMTP-соединения в профилях окружения.

## Критерии приемки (Acceptance Criteria)

### Сценарий 1: Отправка уведомления
* **Given:** Произошло событие (например, документ изменен).
* **When:** Система обработки событий инициирует отправку уведомления.
* **Then:** Пользователь получает уведомление на электронную почту с деталями события.
