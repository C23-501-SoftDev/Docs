# US3.1.2 Фильтрация по диапазону дат

## Описание
Реализация возможности фильтрации результатов поиска по дате создания или изменения документа для сужения выборки.

## Технические требования
* **Стек/Инструменты:** Java/Spring Boot, PostgreSQL.
* **Архитектурные аспекты:** Расширение API поиска параметрами `dateFrom` и `dateTo`. Использование условий SQL `BETWEEN` или операторов сравнения в запросах.
* **Инфраструктура/Конфигурация:** Поля `created_at` и `updated_at` в БД должны иметь индексы для быстрой фильтрации.

## Критерии приемки (Acceptance Criteria)

### Сценарий 1: Успешная фильтрация
* **Given:** В системе есть документы, созданные/измененные в заданном диапазоне дат.
* **When:** Пользователь задает начальную и конечную даты и запускает фильтрацию.
* **Then:** Система возвращает список документов, соответствующих выбранным критериям времени.
