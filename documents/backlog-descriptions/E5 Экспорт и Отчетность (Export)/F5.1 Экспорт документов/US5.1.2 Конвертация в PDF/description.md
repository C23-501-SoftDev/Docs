# US5.1.2 Конвертация в PDF

## Описание
Реализация возможности выгрузки документа в формате PDF для удобного обмена и печати.

## Технические требования
* **Стек/Инструменты:** Java, библиотека для генерации PDF (например, OpenHTMLtoPDF или iText).
* **Архитектурные аспекты:** Генерация PDF на основе ранее сформированного HTML (см. US5.1.1).
* **Инфраструктура/Конфигурация:** Настройка шаблона страницы (ориентация, поля, шрифты) для PDF.

## Критерии приемки (Acceptance Criteria)

### Сценарий 1: Экспорт в PDF
* **Given:** Документ успешно конвертирован в HTML-разметку.
* **When:** Пользователь выбирает экспорт в PDF.
* **Then:** Система генерирует и предоставляет пользователю для скачивания PDF-файл с корректным форматированием.
