# US5.1.3 Конвертация в DOCX

## Описание
Реализация функциональности экспорта документа в формат Microsoft Word (DOCX) для возможности дальнейшего редактирования вне системы.

## Технические требования
* **Стек/Инструменты:** Java/Spring Boot, библиотека Apache POI.
* **Архитектурные аспекты:** Генерация структуры DOCX, маппинг элементов Markdown (заголовки, списки, таблицы) в элементы модели DOCX.
* **Инфраструктура/Конфигурация:** Обеспечение сохранения структуры документа при конвертации из Markdown.

## Критерии приемки (Acceptance Criteria)

### Сценарий 1: Экспорт в DOCX
* **Given:** Документ доступен в системе.
* **When:** Пользователь выполняет запрос на экспорт в DOCX.
* **Then:** Система генерирует файл .docx, открываемый в MS Word с сохранением форматирования документа.
