# US5.1.4 UI: Кнопка Экспорт

## Описание
Внедрение элементов пользовательского интерфейса для инициирования экспорта документа в доступные форматы (PDF, DOCX, HTML).

## Технические требования
* **Стек/Инструменты:** Frontend (HTML, JavaScript).
* **Архитектурные аспекты:** Добавление кнопки/меню "Экспорт" в UI документа. Вызов соответствующих API.
* **Инфраструктура/Конфигурация:** Обработка ошибок при запросе на экспорт (например, если сервер занят или файл поврежден).

## Критерии приемки (Acceptance Criteria)

### Сценарий 1: Отображение UI
* **Given:** Пользователь просматривает документ.
* **Then:** В интерфейсе доступна кнопка "Экспорт" с выпадающим меню выбора форматов (PDF, DOCX, HTML).

### Сценарий 2: Активация экспорта
* **Given:** Меню "Экспорт" открыто.
* **When:** Пользователь выбирает формат (например, PDF).
* **Then:** Система отправляет запрос на сервер и начинает загрузку готового файла браузером.
