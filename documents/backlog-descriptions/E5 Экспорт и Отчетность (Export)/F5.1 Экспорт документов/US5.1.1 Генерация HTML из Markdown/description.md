# US5.1.1 Генерация HTML из Markdown

## Описание
Реализация функциональности преобразования содержимого документа из формата Markdown в HTML для обеспечения корректного отображения и возможности дальнейшего экспорта.

## Технические требования
* **Стек/Инструменты:** Java/Spring Boot, библиотека Markdown-процессор (например, CommonMark или Flexmark).
* **Архитектурные аспекты:** Сервис преобразования контента (Converter Service), который принимает Markdown-текст и возвращает отформатированную HTML-разметку.
* **Инфраструктура/Конфигурация:** Обработка вложений и форматирования при конвертации.

## Критерии приемки (Acceptance Criteria)

### Сценарий 1: Успешная конвертация
* **Given:** Документ в формате Markdown.
* **When:** Пользователь инициирует экспорт в HTML.
* **Then:** Система генерирует HTML-файл с сохраненным форматированием и стилями Markdown.
