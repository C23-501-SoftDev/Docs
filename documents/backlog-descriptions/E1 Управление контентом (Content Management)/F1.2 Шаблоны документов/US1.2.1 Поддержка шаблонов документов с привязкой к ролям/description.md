# US1.2.1 Поддержка шаблонов документов с привязкой к ролям

## Описание
Внедрение системы шаблонов документации, доступность которых определяется ролью текущего пользователя в системе.

## Технические требования
* **Стек/Инструменты:** Java/Spring Boot (Backend), Frontend (UI шаблонов).
* **Архитектурные аспекты:** Хранение шаблонов в системе с привязкой к роли (`Role` -> `List<Template>`). Фильтрация шаблонов при запросе списка доступных.
* **Инфраструктура/Конфигурация:** Шаблоны предзагружены в систему.

## Критерии приемки (Acceptance Criteria)

### Сценарий 1: Доступ к шаблонам
* **Given:** Пользователь авторизован с ролью "Разработчик" (или "Аналитик"/"Администратор").
* **When:** Заходит в режим создания нового документа.
* **Then:** Система отображает только те шаблоны, которые разрешены для данной роли.
