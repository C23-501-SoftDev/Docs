# US1.1.5 Предпросмотр Wiki-разметки в реальном времени

## Описание
Интеграция JavaScript-редактора **Tiptap** в страницу редактирования документа с реализацией двухпанельного интерфейса: слева — поле ввода Markdown/Wiki-разметки, справа — динамический предпросмотр отрендеренного HTML в реальном времени.

## Технические требования
* **Стек/Инструменты:** JavaScript (ES6), Tiptap v2, Prosemirror, HTML, Thymeleaf.
* **Архитектурные аспекты:**
  * Редактор инициализируется на клиенте после загрузки DOM.
  * Исходное содержимое документа передается сервером через встраивание в HTML (`window.__INITIAL_CONTENT__`), что позволяет избежать дополнительного AJAX-запроса при первой загрузке.
  * Двухпанельный интерфейс:
    * **Левая панель:** Активная область редактирования Tiptap с поддержкой Markdown-синтаксиса (заголовки `#`, списки `-`, жирный `**`, код и т.д.).
    * **Правая панель:** Read-only экземпляр Tiptap (или отдельный рендерер), который в режиме реального времени отображает преобразованный HTML из левой панели.
  * Синхронизация: Реализуется через подписку на событие `onUpdate` в Tiptap. Каждое изменение в левой панели вызывает обновление контента в правой панели.
* **Инфраструктура/Конфигурация:**
  * Подключение Tiptap как библиотеки через NPM (с последующей сборкой) или через CDN.
  * Настройка расширений (Extensions) Tiptap: `StarterKit`, `Markdown`, `CodeBlock`, `Image`.

## Критерии приемки (Acceptance Criteria)

### Сценарий 1: Инициализация редактора с существующим текстом
* **Given:** Редактор открывает существующий документ (статус Draft или Published).
* **When:** Страница полностью загрузилась.
* **Then:**
  1. В левой панели отображается Markdown-текст документа (получен из `window.__INITIAL_CONTENT__`).
  2. В правой панели сразу отображается его отрендеренное HTML-представление.

### Сценарий 2: Реактивное обновление предпросмотра
* **Given:** Редактор открыт.
* **When:** Пользователь в левой панели пишет текст `## Заголовок второго уровня`.
* **Then:** В правой панели немедленно появляется заголовок `<h2>Заголовок второго уровня</h2>`, стилизованный согласно CSS системы.

### Сценарий 3: Поддержка Wiki-разметки
* **Given:** Редактор открыт.
* **When:** Пользователь вводит `**жирный** текст` и `* список`.
* **Then:** В правой панели отображается **жирный** текст и маркированный список, соответствующий Wiki-разметке.

### Сценарий 4: Работа с изображениями (Вставка ссылки)
* **Given:** Редактор открыт.
* **When:** Пользователь вставляет ссылку на изображение `![Alt](http://localhost:8080/blobs/123.png)`.
* **Then:** В правой панели отображается превью этого изображения (с сохранением пропорций, вписываясь в ширину панели).
