# US1.3.3 Скачивание вложенного файла

## Описание
Возможность скачивания прикрепленных файлов пользователями, имеющими доступ к документу.

## Технические требования
* **Backend:** Реализация эндпоинта для доступа к файлу по ID вложения.
* **Security:** Проверка прав доступа: скачать файл может только пользователь, имеющий право чтения (READ) пространства, к которому относится документ с этим вложением.

## Критерии приемки (Acceptance Criteria)

### Сценарий 1: Успешное скачивание
* **Given:** Пользователь просматривает документ, содержащий вложение.
* **When:** Пользователь нажимает на ссылку вложения.
* **Then:** Система инициирует загрузку файла на устройство пользователя.

### Сценарий 2: Отсутствие прав
* **Given:** Пользователь пытается скачать файл по прямой ссылке, не имея доступа к документу.
* **When:** Запрос поступает на сервер.
* **Then:** Система возвращает ошибку 403 Forbidden.
