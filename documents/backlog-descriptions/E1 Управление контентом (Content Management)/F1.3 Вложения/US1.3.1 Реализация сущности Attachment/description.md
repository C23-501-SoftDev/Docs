# US1.3.1 Реализация сущности Attachment

## Описание
Реализация поддержки прикрепляемых файлов (вложений) к документам, включая загрузку, хранение (на SMB-шаре) и удаление.

## Технические требования
* **Стек/Инструменты:** Java/Spring Boot, SMB API (интеграция с файловым хранилищем).
* **Архитектурные аспекты:** Сущность `Attachment` (метаданные в БД, ссылка на путь к файлу на SMB).
* **Инфраструктура/Конфигурация:** Права доступа сервисного аккаунта к SMB шаре. Потоковая загрузка/отдача файлов (Multipart).

## Критерии приемки (Acceptance Criteria)

### Сценарий 1: Успешная загрузка
* **Given:** Редактируемый документ.
* **When:** Редактор прикрепляет файл.
* **Then:** Файл сохраняется в целевой SMB-директории, метаданные записываются в БД и связываются с документом.
