# US1.3.2 Удаление вложенного файла

## Описание
Возможность удаления ранее прикрепленных файлов из состава документа для поддержания актуальности данных.

## Технические требования
* **Backend:** API для удаления записи о вложении из БД и физического удаления файла из blob-хранилища.
* **Security:** Проверка прав: удаление доступно пользователю с правами WRITE/OWNER на документ/пространство.

## Критерии приемки (Acceptance Criteria)

### Сценарий 1: Успешное удаление
* **Given:** Документ содержит прикрепленный файл.
* **When:** Редактор нажимает кнопку "Удалить" рядом с вложением.
* **Then:** Система запрашивает подтверждение удаления, после чего файл удаляется из blob-хранилища, а запись о нем — из БД.

### Сценарий 2: Отсутствие прав
* **Given:** У пользователя нет прав на редактирование документа.
* **When:** Пользователь пытается удалить вложение.
* **Then:** Элемент интерфейса (кнопка удаления) скрыт или API возвращает ошибку доступа (403 Forbidden).
