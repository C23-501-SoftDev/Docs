# Участники процессов (Actors)

| Актер | Описание | Основные цели |
| :--- | :--- | :--- |
| **Читатель (Пользователь)** | Любой авторизованный сотрудник, имеющий доступ к системе для ознакомления с документацией. | Поиск информации, чтение документов, экспорт в PDF/DOCX/HTML. |
| **Редактор** | Пользователь с расширенными правами, ответственный за создание и актуализацию контента. | Создание документов по шаблонам, редактирование, управление версиями, настройка доступа к своим документам. |
| **Администратор** | Технический специалист, управляющий всей системой. | Управление пользователями, создание пространств (Spaces), настройка конфигураций сред, восстановление удаленных документов. |
| **Система (System Actor)** | Бэкенд базы знаний, обеспечивающий логику работы приложения. | Обработка запросов, хранение данных, рассылка уведомлений, управление правами доступа. |
