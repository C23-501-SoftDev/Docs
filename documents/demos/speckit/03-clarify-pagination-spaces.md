# Spec Kit: Clarify — вопросы, которые надо согласовать

## Вопрос 1: порядок сортировки
Фиксируем, что порядок “по умолчанию” для контента:
- `createdAt DESC`

Проверка: текущее репозиторное чтение использует `Sort.by("createdAt").descending()`.

## Вопрос 2: поведение при некорректных page/size
Оставляем текущую семантику контроллеров:
- page/size передаются как есть
- при выходе за диапазон: `content` пустой, `totalElements` корректный

## Вопрос 3: где считать totalElements для не-ADMIN
В рамках демонстрации:
- `totalElements` для не-ADMIN = размер множества доступных пространств

Пока допустима реализация через “загрузить + посчитать”, без оптимизации до сложного SQL.
