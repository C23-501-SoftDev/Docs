# Spec Kit: Analyze — что может пойти не так

## Анализ рисков
- Не-ADMIN может потребовать загрузки большого множества `Space` в память для сортировки перед slicing.
- Sorting должен быть детерминированным: если `createdAt` одинаковый, возможны “дрожания” порядка.

## Контроль качества (минимум)
- `totalElements` должен быть одинаковым для всех страниц
- `content` не должен превышать `size`
- при page вне диапазона `content` пустой
