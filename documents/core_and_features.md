# Ядро проекта и Фичи (Core vs Features)

На основе архитектуры, Use Cases и сущностей проекта функционал разделен на минимально жизнеспособный продукт (MVP) и дополнительные расширения.

## 1. Ядро проекта (Core / MVP)
*Минимальный набор функций, необходимый для запуска системы управления базой знаний.*

**Инфраструктура и Безопасность (RBAC):**
* Развертывание Spring Boot бэкенда, PostgreSQL, Git-репозитория (.md файлы) и blob-хранилища.
* JWT-аутентификация (логин/пароль → JWT, хранение токена в HttpOnly Cookie).
* Ролевая модель: глобальные роли `GUEST`, `READER`, `EDITOR` + флаг `is_admin` (Администратор).
* Управление пользователями (CRUD) через панель администратора.
* Назначение прав на уровне пространств (`SpacePermission`) и документов (`DocumentPermission`).

**Структура и Контент (Spaces & Documents):**
* **Пространства (Spaces):** Создание пространств администратором, назначение ответственных и настройка прав доступа.
* **Серверный рендеринг с Thymeleaf** — генерация HTML-страниц на сервере.
* **Tiptap Editor** — двухпанельный WYSIWYG-редактор (Markdown / Preview).
* **Базовый жизненный цикл:** статус `Draft` (Черновик) при создании и `Published` (Опубликован).
* **Навигация:** Иерархическое дерево документов (Оглавление) слева и список на главной странице.

**Интеграция с Git (Базовая):**
* Автоматический коммит (Version) от имени автора при каждом сохранении документа.
* Просмотр истории версий (список: дата, автор, комментарий).

**Поиск:**
* Поиск по заголовкам документов с учетом прав доступа (глобальная роль + права на пространства).

## 2. Фичи (Второстепенные задачи / Расширения)
*Продвинутый функционал, который улучшает пользовательский опыт и автоматизирует процессы (реализуется во вторую очередь).*

**Продвинутая работа с контентом (Epic 1 & 5):**
* **Шаблоны:** Разделение шаблонов по ролям (Разработчик, Аналитик) и автогенерация номеров требований.
* **Вложения:** Drag-and-drop загрузка файлов/картинок в редактор.
* **Экспорт:** Выгрузка документов с сохранением структуры в PDF, DOCX, HTML.

**Продвинутый контроль версий и Жизненный цикл (Epic 2):**
* **Soft-удаление:** Перенос в корзину (изменение статуса на Deleted) и возможность восстановления.
* **Diff и Откат:** Визуальное сравнение двух версий документа (красный/зеленый текст) и функция безопасного отката (создание новой версии из старой).

**Продвинутый поиск и UI (Epic 3 & 4):**
* Фильтрация результатов поиска по дате создания/изменения.
* Системные уведомления (рассылка Email об изменениях в Пространствах).
