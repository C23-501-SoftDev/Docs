## ADDED Requirements

### Requirement: Базовая конфигурация Spring Security
Система ДОЛЖНА защищать все веб-страницы и API-эндпоинты с помощью Spring Security, используя Form Login для аутентификации.

#### Scenario: Перенаправление на страницу входа
- **WHEN** анонимный пользователь пытается получить доступ к защищенному ресурсу
- **THEN** система перенаправляет его на страницу `/login`

### Requirement: Ролевая модель (RBAC)
Система ДОЛЖНА поддерживать роли `ADMIN`, `EDITOR` и `READER`. Информация о пользователях и ролях должна храниться в таблицах `users`, `roles`, `user_roles`.

#### Scenario: Инициализация начальных данных
- **WHEN** база данных инициализируется впервые
- **THEN** в таблицах создаются роли `ADMIN`, `EDITOR`, `READER` и дефолтный пользователь с ролью `ADMIN`
