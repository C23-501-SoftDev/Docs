## ADDED Requirements

### Requirement: Подключение к PostgreSQL
Система ДОЛЖНА поддерживать подключение к базе данных PostgreSQL версии 18.

#### Scenario: Успешное соединение с БД
- **WHEN** приложение запускается с корректными учетными данными БД
- **THEN** соединение устанавливается успешно и пул соединений инициализируется

### Requirement: Управление миграциями БД
Система ДОЛЖНА использовать Liquibase для управления схемой базы данных и версионирования изменений.

#### Scenario: Применение миграций при старте
- **WHEN** приложение запускается на пустой базе данных
- **THEN** Liquibase автоматически применяет все доступные скрипты миграции
