## ADDED Requirements

### Requirement: Базовая структура проекта Spring Boot
Система ДОЛЖНА быть инициализирована как проект Spring Boot с использованием Maven, следуя принципам Clean Architecture (разделение на слои: domain, application, infrastructure, web).

#### Scenario: Успешная сборка проекта
- **WHEN** выполняется команда `mvn clean install`
- **THEN** проект собирается без ошибок и создается исполняемый jar-файл

### Requirement: Стартовая страница приложения
Система ДОЛЖНА предоставлять базовую стартовую страницу (Index Page) по корневому URL (`/`).

#### Scenario: Доступ к главной странице
- **WHEN** пользователь переходит по адресу `/`
- **THEN** отображается приветственная страница приложения

### Requirement: Поддержка профилей окружения
Система ДОЛЖНА поддерживать профили `dev`, `test`, `prod` для разделения конфигураций.

#### Scenario: Запуск с профилем dev
- **WHEN** приложение запускается с параметром `-Dspring.profiles.active=dev`
- **THEN** применяются настройки из `application-dev.yml`
