# Группировка User Stories в Epics и Features

## Epic 1: Управление контентом (Content Management)
**Описание:** Основной функционал по работе с документами, шаблонами и вложениями.

* **Feature 1.1: Редактор документов**
    * US-3: Создание документа по шаблону.
    * Редактирование Wiki-разметки.
    * Автоматическая нумерация требований.
* **Feature 1.2: Шаблоны документов**
    * Набор шаблонов для Разработчика.
    * Набор шаблонов для Аналитика.
    * Набор шаблонов для Администратора.
* **Feature 1.3: Вложения**
    * Загрузка и прикрепление файлов.
    * Управление списком вложений.

## Epic 2: Жизненный цикл и Версионность (Lifecycle & Versioning)
**Описание:** Управление статусами документов и историей их изменений.

* **Feature 2.1: Workflow документа**
    * Статусы: Draft, Published, Deleted.
    * Soft-удаление и восстановление.
* **Feature 2.2: Контроль версий (Git Integration)**
    * US-4: Просмотр истории и откат версий.
    * Визуальное сравнение (Diff).

## Epic 3: Навигация и Поиск (Navigation & Search)
**Описание:** Инструменты для быстрого нахождения информации.

* **Feature 3.1: Поиск и фильтрация**
    * US-1: Поиск по названию и дате.
    * Пагинация результатов.
* **Feature 3.2: Структура и Оглавление**
    * Иерархическое дерево документов.
    * Навигация по Пространствам.

## Epic 4: Администрирование и Безопасность (Admin & Security)
**Описание:** Управление доступом и системными настройками.

* **Feature 4.1: Управление пользователями (RBAC)**
    * Регистрация и авторизация.
    * Назначение ролей (Admin, Editor, Reader).
* **Feature 4.2: Управление Пространствами**
    * US-5: Создание Пространств и настройка прав.
* **Feature 4.3: Системные уведомления**
    * Рассылка Email об изменениях.

## Epic 5: Экспорт и Отчетность (Export)
**Описание:** Выгрузка данных из системы.

* **Feature 5.1: Экспорт документов**
    * US-2: Выгрузка в PDF, DOCX, HTML.
    * Сохранение структуры Wiki-разметки.
